Pure theme - https://github.com/sindresorhus/pure

install

sudo npm install --global pure-prompt
mv ~/.oh-my-zsh/themes/pure.zsh-theme ~/.oh-my-zsh/themes/pure.zsh-theme.bak
ln -s /usr/local/lib/node_modules/pure-prompt/pure.zsh ~/.oh-my-zsh/themes/pure.zsh-theme

Then set theme in ~/.zshrc to pure ZSH_THEME="pure"
